什么是 DreamSpark ？

DreamSpark 是一项 Microsoft 计划，通过允许出于学习、教学和研究的目的来访问 Microsoft 软件，从而为技术教育提供支持。

DreamSpark 很简单：就是为学生免费提供 Microsoft 专业级开发人员和设计人员工具，让学生可以追逐自己的梦想并实现技术上的新突破，或者为其职业开创一个良好的开端。

DreamSpark 帮助教师讲授最新技术和研究实验。 Microsoft 深知，要让学习更能激发现在的学生的积极性、与学生的联系更密切且更能吸引学生的注意，需要借助不同类型的资源。 DreamSpark 为教师提供了一些资源，以确保其课堂总是有最新的技术来迎接挑战、激发学生的积极性并通过新的方式保持学生的注意力。

DreamSpark 也是面向学术机构的订阅：它提供了一种具有成本效益的方式来在实验室和教室以及学生和教工的计算机上配备 Microsoft 开发人员工具、平台和服务器，用于学习和研究的目的。它减少了实验室成本和教学预算。
